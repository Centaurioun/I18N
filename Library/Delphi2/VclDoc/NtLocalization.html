Unit NtLocalization

DescriptionUsesClasses, Interfaces, Objects and RecordsFunctions and ProceduresTypesConstantsVariables

Description

Implements localization routines.

The routines are grouped into following static classes: TNtResource, TNtLocale and TNtMap.

These routines are low level routines. Normally you do not call them but you use TNtLanguageDialog.Select to change a language. However if you want to build your own user interface to select a new language you have to use routines of this unit.

See Samples\Delphi\VCL\CustomSelect or Samples\Delphi\FMX\CustomSelect samples to see how to use the unit.
