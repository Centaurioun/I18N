class function GetAvailable( languages: TNtLanguages; exeFileName: String = ''; compatibleOnly: Boolean = False; checkVersions: Boolean = False): Integer;

Get a list of available resource DLL languages.

Parameters
languages
Returns a list of available languages.
exeFileName
Specifies the name of the application. If empty the current application is used.
compatibleOnly
If True only those resource DLL languages are added that are compatible to the current system settings. Not used in Delphi 2009 or later.
checkVersions
If True only those resource DLL languages are added where the version resource of the DLL matches the EXE.
Returns

Amount of lanugages.

class function ExtractFiles( options: TExtractOptions = [eoCheckDate]; resourceName: String = ''): Integer;

Extracts embedded resource DLLs from EXE to the application directory. If the user has write access to the directory where the EXE locates, then resource files are exteracted there. Otherwise the files are extracted to the user's personal directory.

Parameters
options
Specifies the extract options
resourceName
Specifies the resource type name of the resource DLLs. If empty SOLULING is used.
