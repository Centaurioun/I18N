Unit NtPictureExtension

DescriptionUsesClasses, Interfaces, Objects and RecordsFunctions and ProceduresTypesConstantsVariables

Description

Implements TNtPictureExtension extension class that translates

TPicture (e.g. TImage.Picture).

To enable runtime language switch of images just add this unit into your project or add unit into any uses block. 

implementation

uses
  NtPictureExtension;

See Samples\Delphi\VCL\LanguageSwitch sample to see how to use the unit.
