class function FormatLong( ordinal: TOrdinal; plural: TPlural = pfOne; gender: TGender = geNeutral): String;

Converts an ordinal number to a string using the long format such as "first" and "fourth". Only the first 10 ordinals can be formatted to the long format. The rest of the ordinals will always be formatted using the short format such as "21st" and "24th".

Parameters
ordinal
Ordinal number.
plural
Specifies the plural form.
gender
Specifies the gender.
Returns

The formatted string.
