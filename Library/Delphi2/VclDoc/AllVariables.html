All Variables

Name Unit Description
DefaultLocale NtBase Variable that stores the default language of the application.
FirstDayOfWeek NtBase Variable that stores the first day of the week of the current locale.
FirstWeekOfYear NtBase Variable that stores the first week of the year of the current locale.
LoadedResourceLocale NtBase Currently loaded locale.
NtAfterTranslate NtBaseTranslator Event that is called after a property value has been translated. If you assing this value make sure that the event is as fast as possible because this event is called on every single property of the application.
NtBeforeTranslate NtBaseTranslator An event that is called before translating a property value. Use this to disable or change the translation process. If you assign this value, make sure that the event is as fast as possible because this event is called on every single property of the application.

See Samples\Delphi\VCL\DualLanguage sample to see how to use the event.

NtBeforeTranslateEx NtBaseTranslator An event that is called before translating a property value. Use this to disable or change the translation process. If you assign this value, make sure that the event is as fast as possible because this event is called on every single property of the application.

See Samples\Delphi\VCL\DualLanguage sample to see how to use the event.

NtCheckerExtensions NtChecker Form checker extension. Use this to register your custom checker extensions.
NtEnabledProperties NtBaseTranslator Set of types that are translated. If empty then all properties are translated. If not empty then only those properties whose type is include in this set are translated.

The following sample shows how to translate only string properties. 

unit Unit1;
...
initialization
  NtEnabledProperties := STRING_TYPES;
end.
NtFormChecker NtChecker Form checker. Use this to check your forms.
NtFormPositionTranslationEnabled NtBaseTranslator If True the form position (Left and Top properties) is also translated (e.g. changed if the new resource file contains different Left or Top properties). If False postion is not changed.
NtResources NtResource  
NtTranslatorExtensions NtBaseTranslator Collections of registered extensions. Use it to register your custom extension. 

unit MyExtension;
...
initialization
  NtTranslatorExtensions.Register(TMyExtension);
end.
OperatorDelta NtPattern  
OriginalLanguage NtBase Variable that stores the original language of the application.
PreviouslyLoadedResourceLocale NtBase Previously loaded locale. If empty the current locale is the only locale that has been loaded.
QuotationFixer NtQuotation  
RaiseExceptionOnInvalidPattern NtPattern  
ResourceDllDir NtBase  
ResourceOptions NtBase Resource options.
SystemLanguage NtBase Variable that stores the language of the operating system.
UiLayout NtBase Variable that stores current layout of the user interface.
