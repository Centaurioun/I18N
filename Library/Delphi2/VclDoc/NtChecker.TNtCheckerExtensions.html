Declaration

type TNtCheckerExtensions = class(TNtExtensions)

Description

Class that stores installed checker extension.

If you derive a new extension from TNtCheckerExtension you have to register it in order to take it in use. A good place to register an extension is to call the TNtExtensions.Register in the initialization block of your extension class unit. 

initialization
  NtCheckerExtensions.Register(TYourCheckerExtension);
end.
