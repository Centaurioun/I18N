Unit NtDefault

DescriptionUsesClasses, Interfaces, Objects and RecordsFunctions and ProceduresTypesConstantsVariables

Description

Implements default locale.

If your original language is not English change the value of DefaultLocale from "en" to id your your language or make copy of this unit, add the copy into your project, and change the value of DefaultLocale.

For example if the original language is Japanese change the initialization part to: 

initialization
  DefaultLocale := 'ja';
end.

See Samples\Delphi\VCL\Japanese sample to see how to use the unit.
