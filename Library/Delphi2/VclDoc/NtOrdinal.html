Unit NtOrdinal

DescriptionUsesClasses, Interfaces, Objects and RecordsFunctions and ProceduresTypesConstantsVariables

Description

Implements an ordinal number API.

If you have something like this

str := Format('This is the %d attempt', [attemptCount]);

and you want to localize it convert code to 

resourcestring
  SAttemptCount = 'This is the %s attempt';
...
  str := Format(SAttemptCount, [TNtOrdinal.FormatShort(attemptCount)]);

As you can see instead of passing the attempt count to the Format function we first format the attempt count into string and pass that to Format.

This unit supports following languages: en, fr, de, nl, fi, et, sv, da, no, nb, nn, is, ja, ko, zh-Hans

You can add your own languages by implementing conversion functions and registering them by using TNtOrdinal.Register.

Delphi XE or later required.

Description

Types

TOrdinal = 1..MaxInt;

Ordinal number type.

TSmallOrdinal = 1..10;

Small ordinal number type.

A subset of ordinal numbers containing the first 10 ordinals. When an ordinal number belongs to this subset it also can be written as word. For example "first" vs "1st".

TOrdinalForm = (...);

Enumeration that specifies the string form of an ordinal.

Values
  • ofShort: Short string form such as "1st" and "4th"
  • ofLong: Long string form such as "first" and "fourth"
TOrdinalShortProc = function(ordinal: TOrdinal; plural: TPlural; gender: TGender): String;

Function that returns ordinal in a short form such as "1st" and "4th".
