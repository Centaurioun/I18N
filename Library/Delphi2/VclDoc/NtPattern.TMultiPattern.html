Class TMultiPattern

DescriptionHierarchyFieldsMethodsProperties

Unit

NtPattern

Declaration

type TMultiPattern = class(TObject)

Description

Static class that contains multi pattern (e.g. plural, gender and select) routines.

See also
NtPattern
Implements a multi pattern enabled Format function TMultiPattern.Format that supports plurals, genders and selects.
