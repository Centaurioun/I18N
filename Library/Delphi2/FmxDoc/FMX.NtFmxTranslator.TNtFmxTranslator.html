FMX.NtFmxTranslator

Declaration

type TNtFmxTranslator = class(TNtTranslator)

Description

Class that translates FMX forms.

This class performs runtime language switch by going through each component and property on the host form. Call TNtFmxTranslator.TranslateForms to translate all forms.
