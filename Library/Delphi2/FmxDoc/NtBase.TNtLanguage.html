Methods

class function GetBoth(const native, localize: String): String;

Get language name that contains both native and localized names.

Parameters
native
Language name in the language itself.
localized
Language name in the current language.
Returns

Display name.

class function GetDisplayName( const id: String; locale: Integer = 0; languageName: TNtLanguageName = lnSystem): String;

Get the display name of a language or locale.

Parameters
locale
Windows language or locale id.
languageName
Specifies what kind of name is returned.
Returns

Display name such as English or English (United States).
