Declaration

type TNtExtension = class(TObject)

Description

Class that extends TNtTranslator.

TNtTranslator can translate all trivial properties but can not translate complex properties. For this extensions are used. All extension are derived from TNtExtension class. If you have a 3rd party component that does not translate you have to implement an extension for it. Derive your extension from this class and implement TNtExtension.Translate function.
