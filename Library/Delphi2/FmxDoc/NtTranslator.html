Description

Implements TNtTranslator class that perform runtime language switch.

Normally you don't use this unit. Instead you use NtLanguageDlg.TNtLanguageDialog to perform runtime language switch. If you have your own user interface to select a new language then you use TNtTranslator.SetNew. You can use lower lever routines to select the language and to load a new resource DLL, and to translate all forms into the new language.
